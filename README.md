Django
======

_v0.1 - 20150105_

Docker image with a basic Python [Django](https://djangoproject.com/) install, suitable for use as a Python webapp base image.

### Building ###

Build the Image:

1. `git clone https://github.com/DockerDemos/django.git`
2. `sudo docker build -t django .`
